Getting Started (2 minutes)

1. Choose Your Setup Style

  • 🚀 Express Setup - Quick form, basic schedule
  • 🎯 Guided Setup - Step-by-step with tips and explanations

2. Describe Your Environment

  • Temperature: What's your room temperature? (15-25°C)
  • Radiator: How close is your plant? (This matters more than you think!)
  • Humidity: British homes are typically dry - be honest
  • Light: Actual light levels, not wishful thinking
  • Heating: Scheduled heating creates wet/dry cycles

3. Find Your Plant

Search 120+ common houseplants or browse by category. Each plant shows care requirements and personality.

4. Get Your Schedule

Root Juice calculates a personalised watering timeline based on your home + your plant. No generic advice.

Pro Tips

Environment Accuracy Matters

  • Radiator proximity affects watering frequency by 40%+
  • Scheduled heating creates wet/dry cycles - factor this in
  • North-facing rooms are genuinely darker in winter
  • Humidity is usually lower than you think

Making the Most of Notifications

  • Start with daily reminders - build the habit
  • Use advance warnings if you travel frequently
  • Visual status is key - learn to read the plant indicators
  • One notification per day - we don't spam you

Growth Tracking Workflow

  • Photo before watering - shows actual plant condition
  • Notes capture context - "looking droopy", "new growth spotted"
  • Weekly photos minimum - track meaningful changes
  • Date awareness - timeline shows "3 days ago" vs exact timestamps

Troubleshooting

Notifications Not Working?

  1. Check browser permissions (click 🔔 icon in address bar)
  2. Use "Test Notification" button to verify
  3. Some browsers block notifications in private mode
  4. iPhone: enable in Safari settings

App Feels Slow?

  1. Clear browser cache and reload
  2. Check available storage (Settings → Storage)
  3. Restart browser if issues persist
  4. Export data first if clearing storage

Lost Data?

  1. Check export files in Downloads folder
  2. Data stored locally - clearing browser data removes it
  3. Enable reminders again after data loss
  4. Regular exports prevent this
